Overleaf upload contents for the SETN 2026 draft.

Files:
- `main.tex`: main manuscript file
- `refs.bib`: bibliography
- `article_assets/`: tables and figures used by the manuscript

Overleaf steps:
1. Create a new project in Overleaf.
2. Upload the contents of this folder, or upload the whole folder as a ZIP.
3. Set `main.tex` as the main file if Overleaf does not detect it automatically.

This folder is intentionally slimmed down so it avoids the large benchmark result files from the full repository.
